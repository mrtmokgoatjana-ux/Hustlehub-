# Hustlehub - Money Management App

A Flutter mobile application designed to help users solve money problems through smart budgeting, expense tracking, and financial insights.

## Features
- 💰 Budget Management
- 📊 Expense Tracking
- 💡 Financial Insights
- 🎯 Money Goals
- 📈 Financial Reports

## Getting Started

### Prerequisites
- Flutter SDK (latest version)
- Dart SDK
- IDE: Android Studio, VS Code, or Xcode

### Installation

1. Clone the repository
```bash
git clone https://github.com/mrtmokgoatjana-ux/Hustlehub-.git
cd Hustlehub-
```

2. Get Flutter dependencies
```bash
flutter pub get
```

3. Run the app
```bash
flutter run
```
